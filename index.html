ตั้งค่าเชื่อม Google Sheets

ทำครั้งเดียว ข้อมูลจะเก็บใน Google Sheets ของคุณ

ขั้นตอนที่ 1/4 — สร้าง Google Sheet ใหม่
  1. ไปที่ sheets.new สร้าง Google Sheets ใหม่
  2. ตั้งชื่อว่า "ระบบจองรถ"
  3. คลิก Extensions → Apps Script
ขั้นตอนที่ 2/4 — วางโค้ด Apps Script
  1. ลบโค้ดเดิมทั้งหมดใน Editor
  2. คลิกปุ่มด้านล่างเพื่อคัดลอกโค้ด แล้ววาง (Ctrl+V)
  3. กด Save (Ctrl+S)
  4. เลือก function initSheets แล้วกด Run ▶️
  5. อนุญาต Permission ที่ Google ขอ
✅ คัดลอกแล้ว!
ขั้นตอนที่ 3/4 — Deploy เป็น Web App
  1. คลิก Deploy → New deployment
  2. คลิกไอคอนเฟือง ⚙️ → Web app
  3. Execute as: Me / Who has access: Anyone
  4. คลิก Deploy → อนุญาต Permission
  5. คัดลอก Web App URL
ขั้นตอนที่ 4/4 — วาง Web App URL
